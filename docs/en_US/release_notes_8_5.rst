This release contains a number of bug fixes and new features since the release of pgAdmin 4 v8.5.

**EDB Advanced Server**: 12, 13, 14, 15, and 16

**psql**, **pg_dump**, **pg_dumpall**, **pg_restore**: 16.1


New features

Bug fixes
*********
